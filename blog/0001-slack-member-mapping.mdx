# Map Contributors to Slack Members

You can now go into Settings > Contributors, and map each contributor to their
Slack profile.

![members](/blog-images/slack-member-mapping.png)

This cool because any messages from homie to Slack will now automatically
`@mention` the user to make sure they're notified.
